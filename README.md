你一定遇到过C盘容量告急吧,这个软件就可以帮你拜托这个困境

# 使用方法

下载cDriveSaver.py源文件或到Release下载,直接运行程序,按照提示先输入源文件(夹), 再输入目标文件(夹),即可转移文件,运行时间与文件大小有关

> 建议先做备份,程序为了建造硬链接会删除源文件

软件不会破坏任何程序,使用源路径仍然可用
